# Images Directory

## Required Images

### main-image.jpg
- **Path**: `frontend/public/images/main-image.jpg`
- **Source**: https://github.com/NetAppJpTechTeam/Permission-aware-RAG-FSxN-CDK/blob/master/docker/nextjs/public/images/main-image.jpg
- **Description**: NetApp building with "DATA INFRASTRUCTURE INTELLIGENT" branding
- **Usage**: Used in the sign-in page left panel as the main brand image
- **Dimensions**: Recommended minimum 800x600px
- **Format**: JPEG

## Setup Instructions

1. Download the main-image.jpg from the GitHub repository
2. Place it in this directory (`frontend/public/images/`)
3. The image will be automatically displayed on the sign-in page

## Fallback Behavior

If the image is not found, the component will automatically fall back to a NetApp-branded gradient background.
